Dubito!

A multi-player, online card game.

This real-time online game was made using Django framework. It's websocket-based, which allows for a truly real-time gaming experience. The frontend was made with Vue.js, allowing for reactive updates based on game state.

A D-Risc Assembly compiler and interpreter, entirely written in JavaScript.

This smaller project was inspired by a course I was taking in university. It allows users to write simple programs in a didactic version of Risc Assembly. Such programs are compiled and ran in a JavaScript environment.
